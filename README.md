Zealthy Mini EMR

Technology Stack
----------------
Backend
- Java 21
- Spring Boot
- Spring Data JPA
- PostgreSQL

Frontend
- React
- Vite
- Bootstrap 5
- Axios

Features
--------
Mini EMR
✓ View Patients
✓ Create Patient
✓ Update Patient
✓ Appointment CRUD
✓ Prescription CRUD

Patient Portal
✓ Login
✓ Dashboard
✓ Upcoming Appointments
✓ Upcoming Refills

How to Run
----------
1. Start PostgreSQL
2. Create database zealthy
3. Run Spring Boot
4. Run React
5. Open http://localhost:5173
